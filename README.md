# nmap
FIND NETWORK  VULN/LIVE HOST OR TARGET TO HACK





$ sudo nmap -sP (ip)                                 -Ping ip
$ sudo nmap -Pn                                    -To unblock ping probes
$ sudo nmap -sS (dns or ip)                      -For stealth ping
sudo nmap -sS -p (port) (dns or ip)               -To stealth ping port
